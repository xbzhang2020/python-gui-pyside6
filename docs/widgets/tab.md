# Tab 选项切换

标签页用于切换不同的显示区域。Qt 中可以使用两种方式实现标签页。

- [QTabBar](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QTabBar.html#PySide6.QtWidgets.QTabBar): 用于创建一个选项卡。
- [QTabWidget](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QTabWidget.html#PySide6.QtWidgets.QTabWidget)：用于创建一个标签页。
- [QToolBox](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/QToolBox.html#PySide6.QtWidgets.QToolBox)

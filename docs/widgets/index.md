QtWidgets 提供了一系列创建 UI 的控件，参考 [QWidgets](https://doc.qt.io/qtforpython-6/PySide6/QtWidgets/index.html#module-PySide6.QtWidgets)。

![](https://doc.qt.io/qtforpython-6/_images/parent-child-widgets.png)

以下是一些常用的组件。

- QApplication: 用于创建一个应用实例。
- QLabel
- QPushButton
- QToolButton
- QVBoxLayout
- QHBoxLayout
- QLineEdit
- QDialog
- QTableWidget
- QTableView
- QTreeWidget
- QTreeView
- QListWidget
- QTextEdit
- QMainWindow
- QIcon
- QHeaderView
- QSplitter
